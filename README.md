# Super-Trunfo-em-c-Fundamentos-e-T-cnicas-Avan-adas
Simulador de comparação de cartas estilo Super Trunfo desenvolvido em C. O sistema realiza o cadastro de atributos (população, área, PIB), calcula indicadores como Densidade Populacional e PIB per Capita, e utiliza lógica booleana para determinar a carta vencedora.
